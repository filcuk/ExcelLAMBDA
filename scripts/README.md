# About
## Requirements
- xlwings: `pip install xlwings`

## Scripts
### lambda_report.py
Prints `LAMBDA` functions from Excel workbooks in the `DIRECTORY` and subdirectories.

### add_lambdas.py
Adds specified `LAMBDAS` to Excel workbooks in `DIRECTORY` and subdirectories.
